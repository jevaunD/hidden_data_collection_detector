<h1>Privacy-Aware Autofill Inspector Extension</h1>
Overview
This browser extension enhances user awareness and control by revealing what information websites request through autofill forms. It scans webpage input fields, categorizes data requests based on predefined tags, and clearly displays these to the user — helping increase transparency in everyday online interactions.

Features
Automatically detects webpage input fields tied to autofill data.

Categorizes requested information (e.g., personal, financial, contact data).

Displays data categories and exact form field terms in an easy-to-read panel.

Allows users to dismiss the information panel when not needed.

Designed initially for personal use; open to feedback and security suggestions.

Installation
Clone or download the repository to your local machine.

Open your Chrome or compatible browser.

Navigate to chrome://extensions.

Enable Developer mode (toggle at top right).

Click Load unpacked and select the folder where the extension files reside.

The extension is now installed and ready to use.

Usage
Visit any website with autofill forms.

The extension icon will activate if the page contains detected input fields.

Click the icon to view detailed information about what data fields the site is requesting.

Use the "x" button to close the panel if needed.
